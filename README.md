# 彩虹打字大冒险（Pygame）

一个面向一年级小朋友的彩色打字小游戏，包含三种练习模式，力求色彩柔和、交互简单，降低挫败感，鼓励轻松练习。
\- 大写字母
\- 小写字母
\- 简单拼音（如 ba、bo、ma...）

## 环境要求
- Python 3.8+
- Pygame 2.x

## 安装与运行
```bash
# 1) 安装依赖
pip install pygame

# 2) 运行游戏
python type_game.py
```

## 操作说明
- 菜单界面：按 `1` 进入大写字母、按 `2` 进入小写字母、按 `3` 进入拼音练习。
- 游戏中：直接按键盘对应字符输入；按 `ESC` 返回菜单。
- 目标落出屏幕后不会扣分或结束游戏，尽量保持轻松练习的体验。

## 教学使用方法

### 1. 游戏设计的教育逻辑（Features）
- 模式一：大写字母（认识键盘）
  - 目的：最基础。键盘上的印字通常是大写。让孩子看到屏幕上的 `A` 并在键盘上找到 `A`。
  - 技巧：此时不要求指法，只要求“眼睛找到，手指按下去”。
- 模式二：小写字母（字母转换）
  - 目的：建立 `A = a` 的认知。
  - 挑战：屏幕显示 `a`，孩子需要知道这是键盘上的 `A` 键。这是英语和拼音的基础。
- 模式三：基础拼音（组合概念）
  - 目的：引入“序列”的概念。拼音由声母和韵母组成。
  - 玩法：屏幕出现 `ba`，孩子必须先按 `b` 再按 `a`，强化拼读顺序。

### 2. 物理环境设置（重要！）
- 键盘贴纸：如果键盘是纯黑色，建议用大字体彩色键盘贴纸。
- F 和 J 的定位：教孩子用食指摸到两个小凸起（F、J），告诉他们这是“小房子的门”。
- 遮挡法：拼音练习时尽量不要看键盘，鼓励看屏幕（循序渐进即可）。

### 3. 家长/老师的引导话术
- 不要说：“你打错了。”
- 要说：
  - “哇，你找到了‘B’！它在下面那个角落里。”
  - “这个‘a’是小写的，它的大写朋友在哪里呀？”
  - “我们要打败这个掉下来的‘mo’！先打声母 m，再打韵母 o！”

### 4. 进阶建议（如果你想继续开发）
- 语音读音：当字母出现时，播放“玻(b)”“坡(p)”等声音。
- 奖励机制：每得 100 分放烟花或出现跳舞的小动物。
- 键盘图示：在屏幕下方显示虚拟键盘，高亮当前需要按的键。

这个小游戏代码虽然简单，但包含了核心的“识别-反应-强化”回路，非常适合初学者。

## 常见问题
- 运行时报错 `ModuleNotFoundError: No module named 'pygame'`
  - 解决：`pip install pygame`，或使用虚拟环境后再安装。

## 开发说明
- 如需替换配色、字体大小等，可在 `type_game.py` 中调整 `COLORS`、字号等常量。
- 若要扩展拼音或单词库，修改 `LEVEL_3` 列表即可。
